# Portfolio Website

A personal portfolio website built with Flask, showcasing my projects, GitHub, and LinkedIn. Live at 

## Features
- Built with Python + Flask
- Simple retro-style personal website
- Links to GitHub and LinkedIn
- Visitor counter (optional)

## Run Locally
1. Clone the repo: `git clone https://github.com/YOUR_USERNAME/YOUR_USERNAME.git`
2. Create a virtual environment: `python3 -m venv venv`
3. Activate venv: `source venv/bin/activate`
4. Install dependencies: `pip install -r requirements.txt`
5. Run app: `python app.py`
